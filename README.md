## 알고 모르고 2기 규칙 📋

**기간** : 한 달간 (자율 운영) ~ 5월6일까지 / 5월~ 6월은 같이 풀기
**출처** : [LeetCode Algorithms](https://leetcode.com/problemset/algorithms/)

---

### ✅ 기본 규칙
1. 매일 **1문제 이상** 풀고 인증
2. 인증은 **당일 자정 전**까지
3. 인증 형식 — 문제 링크 + 알게된 내용 또는 팁 (자유)

### 📸 인증 방법
- 풀이 완료 스크린샷 
- 문제 링크도 함께 기재

### 🙏 분위기
- 문제 난이도는 자유 (Easy / Medium / Hard 무관)
- 풀다 막히면 질문 환영, 답 공유도 OK
- 서로 다그치지 않기 — 자율이지만 꾸준히 💪

### ❌ 이런 건 하지 않아요
- 빠진 날 억지로 몰아서 인증 (당일 인증이 의미 있어요)
- 남의 풀이 그대로 복붙 인증
